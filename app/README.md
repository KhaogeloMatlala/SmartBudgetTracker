SmartBudget Tracker

Overview

SmartBudget Tracker is an Android application developed in Kotlin.
It helps users track expenses, manage categories, and monitor monthly spending goals.

Features

- User login screen
- Create and save categories
- Add and save expenses
- Set monthly minimum and maximum goals
- View expenses by selected date range
- View total spending per category
- Local data storage using SQLite database
- Input validation and error handling

Technologies Used

- Kotlin
- Android Studio
- SQLite (SQLiteOpenHelper)
- XML Layouts

How to Run

1. Open the project in Android Studio
2. Build the project
3. Run on emulator or Android device
